# doctor appointment booking 

## 📌 Description
 a full stack doctor appointment booking website . where user can create account , book appointment . admin can add delete doctor and view the appoinment details

## 🚀 Features
- User authentication
- listing doctors

## 🛠️ Tech Stack
- Frontend: React + Tailwind CSS
- Backend: Node.js + Express
- Database: MongoDB
